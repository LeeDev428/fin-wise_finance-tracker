## TL;DR - 4 Commands to Run Your App:

### 1. Install Everything
```powershell
.\setup.ps1
```

### 2. Configure MongoDB
Edit `backend\.env` with your MongoDB Atlas connection string

### 3. Add Sample Data
```powershell
cd backend
npm run seed
```

### 4. Run the App

**Terminal 1 (Backend):**
```powershell
cd backend
npm start
```

**Terminal 2 (Frontend):**
```powershell
cd frontend
npm start
```

Then press `a` for Android or `i` for iOS!
